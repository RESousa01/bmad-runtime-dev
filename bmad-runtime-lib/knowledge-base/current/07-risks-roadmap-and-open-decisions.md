# Risks, Roadmap, and Open Decisions

The deterministic BMAD Help result path is now product-integrated
[KB-CAP-002]. The shortest path to a real AI desktop is therefore production D2
consent and managed brokerage [KB-CLOUD-001]. Before relying on the merged
baseline, requalify the Cargo bootstrap lock digest used by both contract-codegen
authorities. Signing, reproducible packaging, and clean-machine installer
lifecycle evidence remain separate release gates [KB-RELEASE-001].

Web-managed delivery is deferred [KB-ROADMAP-001]. Reopening it requires a new
decision and separate authority implementation; historical dual-delivery plans
do not make it current automatically.
