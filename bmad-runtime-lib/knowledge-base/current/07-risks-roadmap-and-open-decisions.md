# Risks, Roadmap, and Open Decisions

The shortest product path is production D2 consent and managed brokerage,
followed by a real BMAD Help result path [KB-CAP-002][KB-CLOUD-001]. Signing,
reproducible packaging, and clean-machine installer lifecycle evidence remain
separate release gates [KB-RELEASE-001].

Web-managed delivery is deferred [KB-ROADMAP-001]. Reopening it requires a new
decision and separate authority implementation; historical dual-delivery plans
do not make it current automatically.
